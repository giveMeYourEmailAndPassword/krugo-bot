# Шаблон заявки для менеджеров

Минимально: ссылка на договор + что меняем.

```
Заявка на изменение договора
Договор: https://baza.krugo.tours/contracts/ID
Поставщик: СТАРЫЙ → НОВЫЙ
Номер заявки: СТАРЫЙ → НОВЫЙ
Нетто: СТАРОЕ → НОВОЕ
Брутто: СТАРОЕ → НОВОЕ
Остальное не менять
```

## Правила

1. **Договор** — обязательно, полная ссылка
2. **Поставщик** — точное название (ANEX, BEST SERVICE, KOMPAS, PEGAS TOURISTIK, JOINUP, CIP TRAVEL и др.)
3. **Не указано = не меняем**
4. **«Остальное не менять»** — обязательно последней строкой

## Что сейчас работает

| Поле | Что меняется |
|---|---|
| Поставщик | `applications.provider_id` + `contracts.tour_operator` |
| Номер заявки | `applications.number` + `contracts.price_split[0].application` |
| Нетто | `applications.amount` + `contracts.netto_price` |
| Брутто | `contracts.brutto_price` |

## Примеры

### Смена поставщика
```
Заявка на изменение договора
Договор: https://baza.krugo.tours/contracts/t85493bo3ky8ccs
Поставщик: BEST SERVICE → ANEX
Номер заявки: 777777 → 111222
Нетто: 85 → 80
Остальное не менять
```

### Только нетто
```
Заявка на изменение договора
Договор: https://baza.krugo.tours/contracts/t85493bo3ky8ccs
Нетто: 85 → 95
Остальное не менять
```

### Брутто + нетто
```
Заявка на изменение договора
Договор: https://baza.krugo.tours/contracts/t85493bo3ky8ccs
Нетто: 80 → 90
Брутто: 100 → 110
Остальное не менять
```

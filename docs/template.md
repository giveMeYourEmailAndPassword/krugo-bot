# Шаблон заявки для менеджеров

Минимально: ссылка на договор + что меняем.

```
Заявка на изменение договора
Договор: https://baza.krugo.tours/contracts/ID
Поставщик: СТАРЫЙ → НОВЫЙ
Номер заявки: СТАРЫЙ → НОВЫЙ
Нетто: СТАРОЕ → НОВОЕ USD
Брутто: СТАРОЕ → НОВОЕ USD
Отель: СТАРЫЙ → НОВЫЙ
Дата заезда: СТАРАЯ → НОВАЯ
Дата выезда: СТАРАЯ → НОВАЯ
Имя клиента: СТАРОЕ → НОВОЕ
Телефон: СТАРЫЙ → НОВЫЙ
Остальное не менять
```

## Правила

1. **Договор** — обязательно, полная ссылка
2. **Поставщик** — точное название из справочника (ANEX, BEST SERVICE, KOMPAS, PEGAS TOURISTIK, JOINUP, CIP TRAVEL...)
3. **Не указано = не меняем.** Только заполненные строки идут в PATCH
4. Валюта всегда USD, можно не писать
5. Даты: ДД.ММ.ГГГГ
6. «Остальное не менять» — обязательно последней строкой

## Примеры

### Смена поставщика
```
Заявка на изменение договора
Договор: https://baza.krugo.tours/contracts/t85493bo3ky8ccs
Поставщик: BEST SERVICE → ANEX
Номер заявки: 777777 → 111222
Нетто: 85 → 80
Остальное не менять
```

### Смена отеля
```
Заявка на изменение договора
Договор: https://baza.krugo.tours/contracts/t85493bo3ky8ccs
Отель: Armas Labada → Rixos Premium 5⭐
Дата заезда: 26.08.2026 → 01.09.2026
Дата выезда: 26.08.2026 → 08.09.2026
Остальное не менять
```

### Только нетто
```
Заявка на изменение договора
Договор: https://baza.krugo.tours/contracts/t85493bo3ky8ccs
Нетто: 80 → 95
Остальное не менять
```

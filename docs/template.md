# Шаблон заявки для менеджеров

Отправьте в группу krugo-bot. Заполните только нужные строки.

```
Заявка на изменение договора
Договор: https://baza.krugo.tours/contracts/ID

# Поставщики — сколько нужно блоков
Поставщик #1: изменить
  Был: НАЗВАНИЕ
  Стал: НАЗВАНИЕ
  Номер заявки был: XXXXXX
  Номер заявки стал: XXXXXX
  Сумма была: ЧИСЛО
  Сумма стала: ЧИСЛО

Поставщик #2: добавить
  Название: НАЗВАНИЕ
  Номер заявки: XXXXXX
  Сумма: ЧИСЛО

Поставщик #3: удалить
  Был: НАЗВАНИЕ
  Номер заявки был: XXXXXX

# Финансы договора — если меняются
Нетто договора было: ЧИСЛО
Нетто договора стало: ЧИСЛО
Брутто договора было: ЧИСЛО
Брутто договора стало: ЧИСЛО

Остальное не менять
```

## Действия

| Команда | Что делает |
|---|---|
| `изменить` | PATCH существующей заявки |
| `добавить` | POST новой заявки |
| `удалить` | DELETE заявки |

**Идентификация заявки:** для `изменить` и `удалить` нужна пара: **Название поставщика + Номер заявки** (или только номер, если поставщик один). Ищу точное совпадение в `applications`.

## Примеры

### Смена основного поставщика
```
Заявка на изменение договора
Договор: https://baza.krugo.tours/contracts/t85493bo3ky8ccs
Поставщик #1: изменить
  Был: BEST SERVICE
  Стал: ANEX
  Номер заявки был: 777777
  Номер заявки стал: 111222
  Сумма была: 85
  Сумма стала: 80
Нетто договора было: 85
Нетто договора стало: 80
Остальное не менять
```

### Добавить второго поставщика
```
Заявка на изменение договора
Договор: https://baza.krugo.tours/contracts/t85493bo3ky8ccs
Поставщик #2: добавить
  Название: KOMPAS
  Номер заявки: 222222
  Сумма: 45
Остальное не менять
```

### Удалить поставщика
```
Заявка на изменение договора
Договор: https://baza.krugo.tours/contracts/t85493bo3ky8ccs
Поставщик #2: удалить
  Был: KOMPAS
  Номер заявки был: 222222
Остальное не менять
```

### Изменить двух поставщиков + финансы
```
Заявка на изменение договора
Договор: https://baza.krugo.tours/contracts/t85493bo3ky8ccs
Поставщик #1: изменить
  Был: ANEX
  Номер заявки был: 111222
  Номер заявки стал: 333444
Поставщик #2: изменить
  Был: KOMPAS
  Сумма была: 45
  Сумма стала: 50
Нетто договора было: 80
Нетто договора стало: 85
Остальное не менять
```

### Только нетто (поставщики не меняются)
```
Заявка на изменение договора
Договор: https://baza.krugo.tours/contracts/t85493bo3ky8ccs
Нетто договора было: 80
Нетто договора стало: 95
Остальное не менять
```

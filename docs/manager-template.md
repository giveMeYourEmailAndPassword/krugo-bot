# Шаблоны заявок

Выберите нужный, замените значения на свои. Удалите строки, которые не меняются.

---

## Изменить поставщика

```
Заявка на изменение договора

Договор: https://baza.krugo.tours/contracts/t85493bo3ky8ccs

Поставщик #1: BEST SERVICE → ANEX
  Номер заявки: 777777 → 111222
  Сумма: 85 → 80

Нетто: 85 → 80

Остальное — не меняется
```

## Добавить поставщика

```
Заявка на изменение договора

Договор: https://baza.krugo.tours/contracts/t85493bo3ky8ccs

Поставщик #2: добавить KOMPAS
  Номер заявки: 222222
  Сумма: 45

Остальное — не меняется
```

## Удалить поставщика

```
Заявка на изменение договора

Договор: https://baza.krugo.tours/contracts/t85493bo3ky8ccs

Поставщик #2: удалить KOMPAS
  Номер заявки: 222222

Остальное — не меняется
```

## Только финансы

```
Заявка на изменение договора

Договор: https://baza.krugo.tours/contracts/t85493bo3ky8ccs

Нетто: 85 → 95
Брутто: 100 → 120

Остальное — не меняется
```

## Правила

- **Поставщик** — точное название из справочника: ANEX, BEST SERVICE, KOMPAS, PEGAS TOURISTIK, JOINUP, CIP TRAVEL...
- **→** означает «было → стало». Если поставщик не меняется — уберите стрелку
- **добавить** / **удалить** — для новых или удаляемых поставщиков
- **Не включайте поля, которые не меняются** — бот изменит только то, что указано
- **«Остальное — не меняется»** — обязательно в конце

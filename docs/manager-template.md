# Шаблоны заявок

Выберите нужный, заполните, удалите неиспользуемые строки. Отправьте в группу.

---

## Изменить поставщика

```
Заявка на изменение договора
Договор: https://baza.krugo.tours/contracts/t85493bo3ky8ccs
Поставщик #1: изменить
  Был: BEST SERVICE
  Стал: ANEX
  Номер заявки был: 777777
  Номер заявки стал: 111222
  Сумма была: 85
  Сумма стала: 80
Нетто договора было: 85
Нетто договора стало: 80
Остальное не менять
```

## Добавить поставщика

```
Заявка на изменение договора
Договор: https://baza.krugo.tours/contracts/t85493bo3ky8ccs
Поставщик #2: добавить
  Название: KOMPAS
  Номер заявки: 222222
  Сумма: 45
Остальное не менять
```

## Удалить поставщика

```
Заявка на изменение договора
Договор: https://baza.krugo.tours/contracts/t85493bo3ky8ccs
Поставщик #2: удалить
  Был: KOMPAS
  Номер заявки был: 222222
Остальное не менять
```

## Только финансы

```
Заявка на изменение договора
Договор: https://baza.krugo.tours/contracts/t85493bo3ky8ccs
Нетто договора было: 80
Нетто договора стало: 95
Остальное не менять
```

## Правила

- **Замените все значения на свои.** Плейсхолдеров не оставлять
- **Удалите неиспользуемые строки и блоки целиком**
- Договор — обязательно, полная ссылка
- Поставщик — точное название из справочника
- `изменить` / `добавить` / `удалить` — одно действие на блок
- `#1, #2, #3` — нумерация блоков
- «Остальное не менять» — обязательно

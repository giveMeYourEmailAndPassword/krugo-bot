# Шаблоны заявок

Выберите нужный, замените значения на свои. Удалите строки, которые не меняются.

---

## Изменить поставщика

```
Заявка на изменение договора

Договор: https://baza.krugo.tours/contracts/t85493bo3ky8ccs

Поставщик #1: изменить
  Был: BEST SERVICE
  Стал: ANEX
  Номер заявки был: 777777
  Номер заявки стал: 111222
  Цена: 85 → 80
```

## Добавить поставщика

```
Заявка на изменение договора

Договор: https://baza.krugo.tours/contracts/t85493bo3ky8ccs

Поставщик #2: добавить
  Название: KOMPAS
  Номер заявки: 222222
  Цена: 45
```

## Изменить нескольких

```
Заявка на изменение договора

Договор: https://baza.krugo.tours/contracts/t85493bo3ky8ccs

Поставщик #1: изменить
  Был: ANEX
  Номер заявки был: 111222
  Номер заявки стал: 333444

Поставщик #2: добавить
  Название: KOMPAS
  Номер заявки: 222222
  Цена: 45
```

## Финансы договора

```
Заявка на изменение договора

Договор: https://baza.krugo.tours/contracts/t85493bo3ky8ccs

Нетто договора: 80 → 95
Брутто договора: 100 → 120
```

## Правила

- **Поставщик** — точное название: ANEX, BEST SERVICE, KOMPAS, PEGAS TOURISTIK, JOINUP, CIP TRAVEL...
- **изменить** / **добавить** — что сделать с блоком
- **Не указано = не меняется.** Удалите строки и блоки, которые не нужны
- **Цена** — цена поставщика
- **Нетто/Брутто договора** — финансы всей сделки
- Блоки #3, #4 — по необходимости

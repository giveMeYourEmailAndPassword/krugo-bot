# Шаблоны заявок

Выберите нужный, замените значения на свои. Удалите строки, которые не меняются.

---

## Изменить поставщика (только НЕ-approved заявки)

```
Заявка на изменение договора

Договор: https://baza.krugo.tours/contracts/t85493bo3ky8ccs

Поставщик #1: изменить
  Был: BEST SERVICE
  Стал: ANEX
  Номер заявки был: 777777
  Номер заявки стал: 111222
```

## Добавить поставщика

```
Заявка на изменение договора

Договор: https://baza.krugo.tours/contracts/t85493bo3ky8ccs

Поставщик #2: добавить
  Название: KOMPAS
  Номер заявки: 222222
  Цена: 45
```

```
  Был: ANEX

  Название: KOMPAS
  Номер заявки: 222222
  Цена: 45
```

## Клиентский платёж

```
Заявка на платёж

Договор: https://baza.krugo.tours/contracts/t85493bo3ky8ccs

Сумма: 50000
Валюта: KGS
Способ: наличные
Дата: 2026-07-21
Комментарий: аванс от клиента
```

→ Создаёт pending-платёж. Бухгалтер подтвердит в baza.krugo.tours/bookkeeping.

## Оплата поставщику

```
Заявка на оплату поставщику

Договор: https://baza.krugo.tours/contracts/t85493bo3ky8ccs

Поставщик: ANEX
Номер заявки: 111222
Сумма: 4500
Валюта: USD
Тип: полный остаток
```

→ Создаёт pending-запрос на оплату. Бухгалтер выполнит оплату в baza.krugo.tours/bookkeeping.

**Тип:** `полный остаток` (full_remaining) или `аванс` (advance).

```
Поставщик: KOMPAS
```

## Изменение финансов договора

```
Заявка на изменение финансов

Договор: https://baza.krugo.tours/contracts/t85493bo3ky8ccs

Брутто: 100 → 120
Валюта: USD
Причина: доплата за доп. услуги
```

→ Создаёт pending-финанс-запрос. Бухгалтер одобрит в baza.krugo.tours/bookkeeping.

**Одно изменение за заявку.** Для нескольких полей (брутто + нетто) отправьте отдельным сообщением для каждого.

## Правила

- **Поставщик** — точное название: ANEX, BEST SERVICE, KOMPAS, PEGAS TOURISTIK, JOINUP, CIP TRAVEL...
- **изменить** / **добавить** — что сделать с блоком (только для НЕ-approved заявок)
- **Не указано = не меняется.** Удалите строки и блоки, которые не нужны
- **Approved заявки** (finance_status=approved) — нельзя править напрямую. Используйте шаблоны «Корректировка» или «Отмена поставщика».
- **Финансы договора** (нетто/брутто) — через отдельный шаблон «Изменение финансов», не в шаблоне поставщика
- **Валюта** — USD, EUR или KGS
- Блоки #3, #4 — по необходимости

## Недоступно через бот

- **Отмена договора** — требует согласия и файлы. Используйте веб-интерфейс baza.krugo.tours.
- **Подтверждение платежей / одобрение возвратов / корректировок / фин-запросов** — только бухгалтер в baza.krugo.tours/bookkeeping.
